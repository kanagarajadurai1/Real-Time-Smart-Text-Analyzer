# 🚀 Real-Time Smart Text Analyzer

## 📌 Project Overview
The **Smart Text Analyzer** is an interactive web application that allows users to type text and receive **real-time analytics** including character count, word count, sentence count, and remaining characters.

This tool improves writing efficiency by providing **live feedback while typing**.

---

## ✨ Features

🔹 Real-time character counter  
🔹 Remaining character display  
🔹 Maximum character limit (200 characters)  
🔹 Prevents typing beyond the limit  
🔹 Warning message when approaching the limit  
🔹 Word counter  
🔹 Sentence counter  
🔹 Live progress bar indicating usage  
🔹 Copy text functionality  
🔹 Clear text button  
🔹 Responsive and user-friendly interface  

---

## ⚙️ How It Works

1️⃣ The user types text inside the **textarea field**.  
2️⃣ JavaScript listens to the **input event** in real time.  
3️⃣ The system dynamically updates:

- Character count  
- Remaining characters  
- Word count  
- Sentence count  

4️⃣ If the character limit is exceeded, the application **prevents additional typing**.  
5️⃣ The **progress bar visually represents** how much of the character limit is used.
